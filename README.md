# python-stock-analyzer

A simple Python-based stock analysis project that fetches real-time market data, analyzes stock performance, and visualizes trends using graphs.

Features

* Fetch live stock market data
* Analyze historical stock performance
* Visualize stock prices using charts
* Simple and beginner-friendly code structure
* Built using Python and data analysis libraries

Technologies Used

* Python
* Matplotlib
* yFinance

Future Improvements

* Add multiple stock comparison
* Add technical indicators
* Build a web dashboard
* Add stock prediction models
